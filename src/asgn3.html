Grader Notes: This is my Minecraft block world. This world utilizes 5 textures: the grass , the obsidian walls, cobblestone, gold blocks, and the void. The objective of this game is to locate all 20 gold blocks in a minute. The player can move around the world using the arrow keys. There is a pig hidden in the world. (Hint: When Pigs Fly). Extra detials: The "wow" factor of this world is not only did I add an additional game/more textures, but I expirimented with texture blending. The borders of the grass is blended with the void texture to create a dark texture, but still on the same plane. Also, I created the "moon" in the sky as well (Cube).
